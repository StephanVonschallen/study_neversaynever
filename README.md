# study_neversaynever

Code and Materials for the Study "Never Say Never" 

In "Scenarios" you find the different scenarios used for qualitative analysis in Study 1 and quantitative ratings in Study 2 as word documents. 
In "Materials" you find the questionnaire for Study 2. 
In the "Codes" and "Scripts" Folders you find the codes and scripts we used to compare TF-IDF scores of multiple LLM-generated messages for Study 2 Section "Internal and External Model Consistency".
